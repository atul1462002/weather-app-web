🌦️ Weather App

A modern and responsive Weather Application built using HTML, CSS, and JavaScript that fetches real-time weather data using the OpenWeather API.

The app features a beautiful glassmorphism UI, dynamic weather backgrounds, dark/light mode, and a responsive design for all devices.

🚀 Features

✅ Search weather by city name
✅ Real-time weather updates
✅ Temperature, humidity & wind details
✅ 5-Day weather forecast
✅ Dynamic weather backgrounds
✅ Dark / Light mode toggle
✅ Loading animation
✅ Responsive design for mobile & desktop
✅ Smooth UI animations

🛠️ Technologies Used
HTML5
CSS3
JavaScript (ES6)
OpenWeather API
